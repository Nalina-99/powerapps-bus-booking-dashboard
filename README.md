# Responsive Bus Booking Dashboard (Power Apps)

## Overview

This project is a responsive dashboard UI designed using Microsoft Power Apps for a bus booking management system. The dashboard provides a centralized view of booking activity, revenue insights, bus schedules, and operational metrics.

The goal of this project was to design a modern and responsive UI layout using Power Apps components.

---

## Features

### Dashboard Analytics

* Total earnings overview
* Today's booking statistics
* Successful departure metrics

### Bus Management Overview

* List of active buses
* Route and location tracking
* Bus status indicators

### Booking Overview

* Display of recent bookings
* Booking status tracking (Pending / Completed)

### Activity Section

* Upcoming bus departures
* Schedule highlights
* Quick actions for bus management

### Responsive Design

* Dashboard layout optimized for different screen sizes
* Clean sidebar navigation
* Card-based UI design
